Chart Explanation:

  • The red area is the RSS used by VoltDB outside of the JVM, but within the process.
  • Note the red and pink areas are stacked, and together represent RSS.
  • The dotted yellow line represents the amount of memory reserved for the JVM (including memory reserved but unused), plus non-Java memory in use by VoltDB.
    If the JVM were to use all of the memory it was permitted to use, the RSS would reach the yellow line.
